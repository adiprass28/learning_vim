## Options when starting vim

### Advancing to specific place 

**$ vim -c** *<number> file*
buka file pada nomer ke-<number>

**$ vim -c** */pattern file*
buka file pada pattern tersebut 

**$ vim +** file 
buka file pada line terakhir 

### Read-only mode

**$ vim -R** *file*
atau
**$ view** *file*

### Recovering a buffer

#### Recovery in vi

**$ ex -r**
atau 
**$ vi -r**
